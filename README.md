# phpUploader v4.3.5-roflsunriz

## 🎉 概要

phpUploader v4.3.5-roflsunriz は、モダン UI・再開可能アップロード・フォルダ管理・強力なセキュリティ・REST API を備えたファイルアップローダーです。README はインデックスのみを記載し、詳細は各ドキュメントへ分割しました。

![cover](./image/cover.png)

> 注意: `/api/*` を利用するには Web サーバーで `/api/index.php?path=/api/*` へのリライト設定が必要です。詳細は [API.md](docs/API.md) の「付録: ルーティング設定例（Apache / Nginx）」を参照してください。
